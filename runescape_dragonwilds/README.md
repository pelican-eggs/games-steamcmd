# RuneScape: Dragonwilds

RuneScape: Dragonwilds is a co-op survival RPG set in the world of RuneScape. Up to 6 players can explore, craft, build, and battle together on a dedicated server.

## Configuration Parameters / Files

Server settings are stored in `RSDragonwilds/Saved/Config/LinuxServer/DedicatedServer.ini` under the `[/Script/Dominion.DedicatedServerSettings]` section. Pelican manages the values below automatically from server variables on each startup. Manual edits to any key not listed here (notably `ServerGuid`, which the game generates on first boot) are preserved across restarts.

```ini
;METADATA=(Diff=true, UseCommands=true)
[SectionsToSave]
bCanSaveAllSections=true
[/Script/Dominion.DedicatedServerSettings]
OwnerId=
ServerName=
DefaultWorldName=
AdminPassword=
WorldPassword=
MaxPlayers=6
AdministratorList=()
ServerGuid=
```

|Key|Required|Default|Description|
|---|---|---|---|
|OwnerId|Yes| |Your RuneScape: Dragonwilds Player ID. Found in-game at the bottom of the Settings menu.
|ServerName|Yes| |Name shown in the in-game server browser.
|DefaultWorldName|Yes| |Name of the world created on first startup. Players search by this name in the Public Worlds tab (case-sensitive).
|AdminPassword|Yes| |Password that grants admin status when entered via Pause > Settings > Server Management.
|WorldPassword|No| |Join password. Leave empty for a public server.
|MaxPlayers|No|6|1–6 players. RAM requirement is ~2 GB base + 1 GB per additional player.
|AdministratorList|No| |Comma-separated Player IDs granted permanent admin access. Include parentheses: (12345,67890).
|ServerGuid|Auto| |Generated by the game on first boot. Do not modify.

### Sample Ignore File for Backups

World saves are the only data critical to a full restore — game binaries are redownloaded by Steam and the config file is repopulated by Pelican on next startup. PersistentDownloadDir and SpudCache are excluded as they are large and fully regenerated.
```
# Ignore all game files
*
# Keep save data
!RSDragonwilds/
!RSDragonwilds/Saved/
!RSDragonwilds/Saved/SaveGames/
!RSDragonwilds/Saved/Config/
!RSDragonwilds/Saved/Logs/
# Exclude large/unnecessary Saved subdirectories
RSDragonwilds/Saved/PersistentDownloadDir/
RSDragonwilds/Saved/SpudCache/
```
