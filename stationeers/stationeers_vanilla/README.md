# Stationeers

Construct and manage your own space station either by yourself in singleplayer or with friends online! Fully functioning atmospherics, science, power, engineering, medical, logic, and agricultural systems. Explore to find asteroids and construct elaborate factories to harvest your resources!

## SPECIAL NOTE 

- **Console Input:** Sending commands directly through the console is currently not supported by the dedicated server. THIS IS NOT AN ERROR IN THIS EGG!

## Server Information

- **DRM-Free & Independent:** Since June 2022, Stationeers is entirely DRM-free and no longer requires Steam to run. The dedicated server uses RocketNet (a custom RakNet implementation) for high-performance direct connections
- **Networking:** While the game supports Steam P2P for local hosting, this dedicated server relies on Direct Connect. This simplifies the networking stack to a single port
- **Steam Integration:** Even without DRM, the server registers with the developers' central discovery server. This allows social features like "Join Game" via the Steam Friend List to function normally when players are using the Steam client

## Server Ports

Stationeers requires only 1 port

| Port        | default |
|-------------|---------|
| Game        | 27016   |

### Notes

- **Port Flexibility:** 27016 is the default, but any port can be used
- **Visibility:** The server registers itself with the master server using the specified port. This ensures the dedicated server remains visible in the in-game server list regardless of the port number chosen
