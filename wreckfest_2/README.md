## Server Config
Additional server options are available in the server config file `/save/server_config.scnf`. Server name, password, and game port are managed by the egg variables/default allocation and are written into the config by the panel. Use the config file for gameplay-facing settings such as server description, moderation settings, event loop settings, cars, tracks, and rotation behavior.

Another file will be generated on first start at `/save/default_loop.becl`. This file contains options for each map such as weather, game mode, flags, reset delay, and number of bots.
