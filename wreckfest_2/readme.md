## Server Config
Addition server options are available in the server config file `/defaults/server_config.scnf`. You can change your server description, how race cups are handled, and starting orders.

Another file will be generated on first `/defaults/default_loop.becl`. This file contains options for each map such as weather, game mode, flags, reset delay, and number of bots.
